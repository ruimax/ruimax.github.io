css动画与js动画
css3 transform(变形) css3 transition(过渡) css3 animation(逐帧动画)
transform
  • 旋转rotate: transform: rotate(30deg)
  • 扭曲skew(翻转): skew(30deg,20deg)
  • 缩放scale: scale(2,4) 宽度转换为原始尺寸的 2 倍，高度转换为原始高度的 4 倍
  • 移动translate: translate(50px,100px)
  • 矩阵变形matrix: 把所有 2D 转换方法组合在一起，旋转、缩放、移动以及倾斜元素
transition过渡
  • transition: <property> <duration> <timing-function> <delay>;
  • transition:width 2s, height 2s, background-color 2s, transform 2s;
animation动画
  • animation: @keyframes duration | timing-function | delay
  • animation: myFrames 3s ease-in 1s
  • @keyframes myFrames{
        from {background: red;}
        to {background: yellow;}
    }
  • @keyframes myFrames{
        0% {background: red;}
        25% {background: yellow;}
        50% {background: green;}
        75% {background: blue;}
        100% {background: black;}
    }
